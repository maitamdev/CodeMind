BÀI 1: PYTHON LÀ GÌ? DÙNG ĐỂ LÀM GÌ?
1.1. Python là gì?

Python là một ngôn ngữ lập trình bậc cao, được thiết kế để dễ đọc, dễ viết và dễ học.

Nói đơn giản, Python là một công cụ giúp con người ra lệnh cho máy tính thực hiện một công việc nào đó.

Ví dụ:

print("Xin chào Python")

Dòng lệnh trên yêu cầu máy tính in ra màn hình dòng chữ:

Xin chào Python

Python được tạo ra bởi Guido van Rossum và ra mắt lần đầu vào năm 1991. Tên gọi Python không xuất phát từ loài rắn, mà được lấy cảm hứng từ chương trình hài “Monty Python’s Flying Circus”.

1.2. Ngôn ngữ lập trình là gì?

Máy tính không tự hiểu ngôn ngữ con người như tiếng Việt hoặc tiếng Anh. Máy tính chỉ hiểu các chỉ dẫn rất cụ thể.

Ngôn ngữ lập trình là phương tiện để con người giao tiếp với máy tính.

Ví dụ, thay vì nói:

Máy tính ơi, hãy cộng 5 với 3 rồi in kết quả ra màn hình.

Ta viết bằng Python:

print(5 + 3)

Kết quả:

8
1.3. Vì sao Python dễ học?

Python được đánh giá là một trong những ngôn ngữ lập trình dễ học nhất vì cú pháp của nó gần giống tiếng Anh và ít ký hiệu phức tạp.

So sánh đơn giản:

Python
print("Hello")
Java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello");
    }
}

Với cùng một việc là in chữ “Hello”, Python ngắn gọn hơn rất nhiều.

Điều này giúp người mới tập trung vào tư duy lập trình thay vì bị rối bởi cú pháp phức tạp.

1.4. Python dùng để làm gì?

Python được sử dụng trong rất nhiều lĩnh vực khác nhau.

1.4.1. Lập trình web

Python có thể dùng để xây dựng website, hệ thống quản trị, API và backend server.

Một số framework phổ biến:

Django
Flask
FastAPI

Ví dụ ứng dụng:

Website bán hàng

Trang blog cá nhân

Backend cho ứng dụng thương mại điện tử
1.4.2. Tự động hóa công việc

Python rất mạnh trong việc tự động hóa các công việc lặp đi lặp lại.

Ví dụ:

Đổi tên hàng trăm file cùng lúc
Đọc dữ liệu từ Excel
Gửi email tự động
Tải file từ internet
Tự động tạo báo cáo
Tự động kiểm tra dữ liệu

Ví dụ đơn giản:

import os

print(os.listdir())

Đoạn code trên liệt kê các file trong thư mục hiện tại.

1.4.3. Phân tích dữ liệu

Các thư viện thường dùng:

NumPy

Matplotlib

Ví dụ ứng dụng:

Phân tích doanh thu bán hàng

Vẽ biểu đồ báo cáo
Dự đoán xu hướng kinh doanh
1.4.4. Trí tuệ nhân tạo và Machine Learning

Các thư viện nổi bật:

Backend là phần xử lý phía sau của một ứng dụng.

Ví dụ khi người dùng đăng nhập vào website, backend sẽ:

Python có thể dùng để xây dựng backend bằng Django, Flask hoặc FastAPI.

1.4.6. Scripting

Script là các chương trình nhỏ dùng để xử lý một công việc cụ thể.

Ví dụ:

Python rất phù hợp để viết script vì cú pháp ngắn gọn, dễ hiểu và có nhiều thư viện hỗ trợ.

1.5. Ưu điểm của Python

Python có nhiều ưu điểm:

Dễ học

Cú pháp gần với ngôn ngữ tự nhiên.

if age >= 18:
    print("Bạn đã đủ tuổi")

Đọc gần giống như tiếng Anh:

Nếu tuổi lớn hơn hoặc bằng 18 thì in ra “Bạn đã đủ tuổi”.

Dễ đọc

Code Python thường gọn và rõ ràng.

name = "An"
age = 20

print(name)
print(age)
Có cộng đồng lớn

Khi gặp lỗi, người học rất dễ tìm được tài liệu, video, bài viết hoặc câu trả lời trên internet.

Nhiều thư viện

Python có hàng nghìn thư viện hỗ trợ sẵn.

Ví dụ:

import math

print(math.sqrt(16))

Kết quả:

4.0
Ứng dụng rộng

1.6. Nhược điểm của Python

Python không hoàn hảo. Nó cũng có một số nhược điểm:

Tốc độ chậm hơn C/C++

Python thường chậm hơn các ngôn ngữ như C, C++ vì Python là ngôn ngữ thông dịch.

Không phải lựa chọn tốt nhất cho mobile app

Python không phổ biến bằng Kotlin, Swift hoặc Flutter trong phát triển ứng dụng mobile.

Quản lý bộ nhớ không trực tiếp

Người học không cần quản lý bộ nhớ thủ công, điều này giúp dễ học nhưng cũng khiến người học ít hiểu sâu về bộ nhớ máy tính ở giai đoạn đầu.

Tuy vậy, đối với người mới học lập trình, Python vẫn là một lựa chọn rất tốt.

1.7. Python hoạt động như thế nào?

Người học viết code Python
        ↓
File .py
        ↓
Python Interpreter đọc code
        ↓
Máy tính thực hiện lệnh
        ↓
Hiển thị kết quả

BÀI 2: CÀI ĐẶT PYTHON, VS CODE, PYCHARM
2.1. Vì sao cần cài đặt môi trường lập trình?

Để viết và chạy chương trình Python trên máy tính, ta cần có:

Python Interpreter
Đây là chương trình giúp máy tính hiểu và chạy code Python.
Trình soạn thảo code
Đây là nơi ta viết code, ví dụ VS Code hoặc PyCharm.
Terminal hoặc Command Prompt
Đây là nơi ta gõ lệnh để chạy chương trình.
2.2. Cài đặt Python
Bước 1: Tải Python

Truy cập trang chính thức của Python:

python.org

Chọn phiên bản Python mới nhất phù hợp với hệ điều hành.

Bước 2: Cài đặt Python

Khi cài đặt trên Windows, cần chú ý tích vào ô:

Add Python to PATH

Đây là bước rất quan trọng.

Nếu không tích, máy tính có thể không nhận lệnh python trong Terminal.

Bước 3: Kiểm tra cài đặt

Mở Terminal hoặc Command Prompt và gõ:

python --version

Hoặc:

python3 --version

Nếu hiển thị phiên bản như:

Python 3.12.x

nghĩa là đã cài đặt thành công.

2.3. Python PATH là gì?

PATH là danh sách các vị trí mà hệ điều hành sẽ tìm kiếm chương trình khi ta gõ lệnh.

Khi ta gõ:

python

Máy tính sẽ tìm trong PATH xem chương trình Python nằm ở đâu.

Nếu Python chưa được thêm vào PATH, máy tính có thể báo lỗi:

'python' is not recognized as an internal or external command

Lỗi này thường gặp trên Windows.

BÀI 3: CHẠY FILE .PY, PYTHON SHELL, TERMINAL
3.1. File .py là gì?

File .py là file chứa mã nguồn Python.

Ví dụ:

main.py

Bên trong file có thể chứa:

print("Đây là file Python")

Khi chạy file này, Python sẽ đọc và thực hiện các lệnh bên trong.

3.2. Cách chạy file Python

Giả sử có file:

hello.py

Nội dung:

print("Xin chào")

Ta có thể chạy bằng lệnh:

python hello.py

Hoặc trên một số máy:

python3 hello.py

Hoặc Windows có thể dùng:

py hello.py
3.3. Terminal là gì?

Terminal là công cụ cho phép người dùng giao tiếp với máy tính bằng lệnh.

Trên Windows, Terminal có thể là:

Command Prompt
PowerShell
Windows Terminal

Trên macOS:

Terminal

Trên Linux:

Terminal

3.5. Python Shell là gì?

Python Shell là môi trường cho phép gõ lệnh Python và xem kết quả ngay lập tức.

Mở Terminal và gõ:

python

Hoặc:

python3

Khi vào Python Shell, ta thấy dấu nhắc:

>>>

Ví dụ:

>>> 2 + 3
5

Hoặc:

>>> print("Hello")
Hello
3.6. Khi nào dùng Python Shell?

Python Shell phù hợp để:

Thử nhanh một dòng lệnh
Kiểm tra phép tính

Học cú pháp cơ bản

Ví dụ:

>>> 10 * 5

>>> "Python".upper()
'PYTHON'

4.1. Một chương trình Python gồm những gì?

Comment

4.2. Ví dụ chương trình đơn giản
# Chương trình đầu tiên của tôi

Trong đó:

# Chương trình đầu tiên của tôi

là comment.

Hai dòng sau là lệnh in ra màn hình.

4.3. Comment là gì?

Comment là phần ghi chú trong code. Python sẽ bỏ qua comment khi chạy chương trình.

Comment giúp:

Giải thích code
Ghi chú mục đích chương trình
Giúp người khác đọc hiểu code
Giúp chính bạn hiểu lại code sau này

Ví dụ:

# In ra lời chào
print("Xin chào")

Dòng bắt đầu bằng dấu # là comment.

4.4. Comment một dòng
# Đây là comment một dòng
print("Hello")
4.5. Comment nhiều dòng

Python không có cú pháp comment nhiều dòng chính thức như một số ngôn ngữ khác, nhưng có thể dùng nhiều dòng #:

# Đây là chương trình minh họa
# Tác giả: Sinh viên A
# Mục tiêu: Làm quen với Python

print("Hello Python")

Hoặc dùng chuỗi nhiều dòng:

"""
Đây là phần mô tả chương trình.
Chương trình này dùng để minh họa Python cơ bản.
"""

print("Hello")

Tuy nhiên, với người mới học, nên dùng dấu # cho ghi chú thông thường.

5.0

math là thư viện toán học.
sqrt() là hàm tính căn bậc hai.

Code đúng:

4.10. Thụt lề trong Python

Python sử dụng thụt lề để xác định khối lệnh.

Ví dụ:

age = 20

if age >= 18:
    print("Đủ tuổi")

Dòng:

    print("Đủ tuổi")

được thụt vào bên trong if.

Nếu viết sai thụt lề, chương trình có thể báo lỗi.

Ví dụ sai:

age = 20

if age >= 18:
print("Đủ tuổi")

Lỗi vì dòng print() không được thụt vào.

4.11. Quy tắc thụt lề

Thông thường, Python sử dụng 4 dấu cách cho một cấp thụt lề.

Nên dùng:

if True:
    print("Đúng")

5.1. Học lập trình khác học lý thuyết thông thường

Lập trình là kỹ năng thực hành. Không thể giỏi lập trình chỉ bằng cách đọc sách hoặc xem video.

Để học tốt lập trình, người học cần:

Đọc lý thuyết
Viết code
Gặp lỗi
Sửa lỗi
Làm bài tập
Làm dự án
Lặp lại nhiều lần

Học lập trình giống học lái xe hoặc học chơi nhạc cụ. Chỉ xem người khác làm không đủ. Phải tự tay thực hành.

5.2. Sai lầm phổ biến khi học lập trình
Sai lầm 1: Chỉ xem video, không tự code

Nhiều người học xem video và cảm thấy đã hiểu. Nhưng khi tự viết lại thì không biết bắt đầu từ đâu.

Cách khắc phục:

Xem đến đâu, code đến đó.
Không copy toàn bộ code.
Sau khi xem xong, tự viết lại không nhìn tài liệu.
Sai lầm 2: Sợ lỗi

Lỗi là một phần bình thường của lập trình.

Lập trình viên chuyên nghiệp cũng gặp lỗi mỗi ngày.

Thay vì sợ lỗi, người học nên tập đọc lỗi.

Ví dụ lỗi:

SyntaxError: missing closing parenthesis

Nghĩa là thiếu dấu ngoặc đóng.

Sai lầm 3: Học quá nhiều lý thuyết trước khi thực hành

Một số người học cố học hết mọi thứ rồi mới làm bài tập. Cách này dễ gây chán và khó nhớ.

Cách tốt hơn:

Học một ít → thực hành ngay → sai → sửa → hiểu sâu hơn
Sai lầm 4: Nhảy quá nhanh sang chủ đề khó

Người mới thường muốn học AI, web hoặc automation ngay. Nhưng nếu chưa chắc nền tảng, khi vào dự án thật sẽ rất dễ bị rối.

KẾT LUẬN CHƯƠNG 1

Chương 1 là bước khởi đầu quan trọng trong toàn bộ khóa học Python. Mục tiêu không phải là học nhiều cú pháp phức tạp, mà là giúp người học hiểu Python là gì, cài đặt được môi trường lập trình, chạy được chương trình đầu tiên và hình thành tư duy học lập trình đúng cách.

Sau chương này, người học chưa cần viết được chương trình phức tạp, nhưng phải tự tin thực hiện được các việc sau:

Mở VS Code.
Tạo file Python.
Viết vài dòng lệnh print().
Chạy chương trình.
Đọc lỗi cơ bản.
Hiểu rằng lập trình là quá trình thực hành, sai, sửa và tiến bộ dần.
